## 🏠 Real Estate & Home Services

Servers for real estate data, home services, property operations, cleaning, housing workflows, or related booking and advisory tasks.

- [Zillapi Zillow Property Data](https://github.com/nikhonit/zillow-skills): Zillow property data MCP server with 4 tools for address lookup, ZPID lookup, Zestimate retrieval, and active listing search. Returns Zestimates, rent estimates, tax records, listing details, price history, school data, photos, and 300+ property fields. Streamable HTTP endpoint: https://api.zillapi.com/mcp. OAuth 2.1 with PKCE. MIT-0.
- [GetColby/claw-cleaning](https://github.com/GetColby/claw-cleaning): Remote Streamable HTTP MCP server for AI-booked apartment cleaning in San Francisco. Books weekend cleanings end-to-end, priced at $40/hour, with 3 tools for quoting and scheduling. Endpoint: `https://claw.cleaning/mcp`. No auth. MIT.
- [USDV Capital Real Estate CFO](https://smithery.ai/servers/usdv-capital/real-estate-cfo): Real estate market intelligence, financial calculators, and capital advisory MCP covering all 50 US states plus DC and Puerto Rico. 23 tools for market search, location statistics, comparisons, DSCR-style calculators, eligibility checks, and advisory workflows. Smithery install: `npx -y smithery mcp add usdv-capital/real-estate-cfo`; direct endpoint: `https://real-estate-cfo--usdv-capital.run.tools`.
